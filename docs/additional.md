# Technical Metadata

The new ingest system enables depositors to utilise a single metadata template to upload their files to the system. However, for some data types additional technical metadata is required. Below is a full list of the file types that require an additional metadata file to be included.

## How to upload your technical metadata

The additional technical metadata files should be included in the ingest system alongside the other files to be deposited. It is not required to list these additional metadata as a sperate row in the template, however, each file (with approproate file extension) should be listed in the __Additional Metadata__ column of the [metadata template](./metadata.md). If you fail to include  requred additinal metadata file or include the file but do not list it in the metadata templete, then the Ingest will produce an error.
